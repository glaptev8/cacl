# КАЛЬКУЛЯТОР #<br/>

данный калькулятор поддерживает выражения содержащие следующие операции:<br/>
"+-*/()"<br/>
также калькулятор поддерживает переменные:\n<br/>
n = 5<br/>
d = 10<br/>
c = d<br/>
(n + d) * c + 20 / 10<br/>
<br/>
запустить:<br/>
git clone https://github.com/glaptev8/calc && cd calc/srcs && java Main.java<br/>
